Owngar
======

Owngar is a client-side [react-canvas](https://github.com/Flipboard/react-canvas)
clone of the worldwide sensation of [Agar.io](http://agar.io/)!

Installation
------------

```bash
$ npm install -g bower http-server webpack
$ npm install
$ bower install
```

Deployment
------------

```bash
$ webpack --watch
$ http-server
$ open http://localhost:8080
```
